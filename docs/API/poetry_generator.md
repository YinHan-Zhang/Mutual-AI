# 古诗词生成

> http://ai.9998k.cn/api/poetry_generator

*请求方式*：GET

正文参数：

| 参数名      | 类型 | 内容     | 必要性 | 备注 |
| :---------- | ---- | -------- | ------ | ---- |
| type_choice | num  | 生成类型 | 必要 | 1为自动生成<br />2为由首局生成<br />3为藏头诗 |
| addition | str | 关键词 | 当`type_choice`不为1时必要 | 生成关键词 |

json回复：

根对象：

| 字段 | 类型 | 内容     | 备注 |
| ---- | ---- | -------- | ---- |
| res  | str  | 返回内容 |      |
